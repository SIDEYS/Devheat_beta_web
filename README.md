How to use our website:-
1) First open the terminal in vs code and write npm start 
2)The server will run at Port 3000
3)In any browser type localhost:3000
And it will direct you to the landing page of the the website 
4)Click on menu in the navbar it will direct you to the menu page.
5) Order the food you like and then click on "generate bill".
6) A bill will be automatically generated.
7)It will have all the items and the quantity of each item and a total amount of the products with added 18% gst.
